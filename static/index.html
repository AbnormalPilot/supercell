OpenEnv v1 Compliance

Built strict.

Pydantic models, dense reward, deterministic graders, canonical endpoints. Passes openenv validate locally and at runtime.

01

Typed Pydantic models

Action, Observation, and State inherit from openenv.core.env_server.types. /schema returns all three.

02

Dense reward signal

+10 base, +25 MAYDAY, +10 medical, +15 fuel-save, up to +10 passenger bonus, −100 per crash, +50 perfect episode.

03

Deterministic graders

Every task has a pure function from landing and crash logs to a score in [0, 1]. Same episode, same score — always.

04

OpenAI inference.py

Strict [START] / [STEP] / [END] stdout. Reads API_BASE_URL / MODEL_NAME / HF_TOKEN.

05

Docker & HF Space

Single-stage Python 3.12 image on port 7860. Runs inside the 2 vCPU / 8 GB hackathon budget.

06

Canonical endpoints

/reset, /step, /state, /schema, /mcp, /health, /metadata, /ws — all wired by OpenEnv core.
